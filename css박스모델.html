크기

크기는 size 속성으로 변경할수 있고 기본크기는 p태그의 글자크기로 동일하다.
h1 태그의 글자는 이미 200%이 된 크기로 출력이되고 있고
동일한 font-size를 넣어주면 다른 글자들도 동일한 크기가 된다

box-size border-box 변경

box-size를 border-box로 변경을 하고 작성하면 height/width의 값이 border+padding+content 값으로 바뀐다
크기값이 중요한 경우에는 border-box로 바꿔서 작성 크기 계산이 편하기 때문

마진값-auto

margin 속성값 auto

margin 속성값 auto값을 사용해줄때는 display가 block속성, content 크기가 부모보다 작아야한다.
(부모 크기와 같으면 마진이 들어갈 공간이 없다)
margin - auto를 사용하면 부모의 크기 기준으로 중앙으로 정렬된다.

아웃라인 - outline

아웃라인입니다

outline은 border와 동일하게 사용할 수 있다
그려지는 위치도 동일하게 그려진다
단, 두께는 크기의 범위에 들어가지 않기 떄문에 계산 할 필요가 없다

outline과 box-size에서 border-size로 작성했을때,
*outline을 사용하면 border값 계산 x
*box-size를 사용하면 padding과 content(height/width)를 따로 계산 할 필요없음
> content 값만 계산 >> 요소의 크기 계산 : margin + content
